101 LINQ Samples in Clojure
===========================

Port of [C#'s' 101 LINQ Samples](http://code.msdn.microsoft.com/101-LINQ-Samples-3fb9811b) translated into Clojure.  

Execute and display the results of all the examples with:

    lein run
